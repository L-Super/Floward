# Floward

/fləʊ-wərd/ （近似 Forward）

一个基于 Qt 的跨平台剪贴板管理工具，支持多设备同步。可以从剪贴板历史记录中选择对应的项进行粘贴。它提供了一种查看剪贴板历史记录和快速粘贴任何以前复制的内容的简便方式。

![](images/screenshot2.png)

## 特性

- 📋 本地剪贴板管理
- 🔄 多设备数据实时同步（文本、图片）
- 🖼️ 图片自动同步下载和剪贴板更新
- ⚡ 全局快捷键支持，支持自定义（默认`Alt + V`）
- 🎯 系统托盘运行

## 适用场景

Floward 特别适合以下场景：

- 💻 **多设备办公**：在台式机复制代码片段，在笔记本上直接粘贴使用
- 🎨 **设计工作流**：在一台设备复制图片，另一台设备立即可用，无需手动传输
- 📝 **远程办公**：在不同位置的设备间无缝共享剪贴板内容
- 🔐 **数据安全**：通过自建服务器，确保敏感数据不经过第三方服务

## 版本说明

[Releases](https://github.com/L-Super/Floward/releases) 有两个版本，区别如下：

| 版本   | 描述                                   |
| ------ | -------------------------------------- |
| 单机版 | 纯本地。数据保留在设备上。无需互联网。 |
| 同步版 | 支持多设备同步。在不同设备间同步数据。 |

## 构建

构建要求：

- C++ 20
- Qt 6.0+
- CMake


```bash
cmake -B build
cmake --build build
```

## 客户端配置

本应用需要同步服务器才能实现多设备同步功能。您可以使用 [ClipboardServer](https://github.com/L-Super/ClipboardServer) 部署自己的服务器。关于服务器的部署和配置，请参考 ClipboardServer 项目的详细说明。

### 配置同步服务器

**方法一：**

1. 运行 Floward
2. 打开同步服务器 web 服务地址
3. 输入账户信息登录

**方法二：**

1. 运行 Floward，打开 Floward 应用设置 - 同步
2. 填写服务器地址，如：`https://your-domain.com`
3. 点击登录，此时会打开 Web 页面
4. 输入账户信息登录

配置完成后，应用将自动连接到服务器并开始同步剪贴板数据。
